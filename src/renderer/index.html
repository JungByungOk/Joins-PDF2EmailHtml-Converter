설정 도움말
렌더링 DPI
PDF를 이미지로 변환할 때의 해상도입니다. 높을수록 선명하지만 파일 크기가 증가합니다. 일반 문서는 150~200, 세밀한 이미지가 포함된 문서는 250 이상을 권장합니다.
이미지 배율
원본 이미지 대비 출력 이미지의 크기 비율입니다. 50%이면 원본의 절반 크기로 출력됩니다. 이메일 본문에 적합한 크기로 조정하세요.
여백 줄이기
페이지 상단/하단/내부의 불필요한 여백을 자동으로 감지하여 줄여줍니다. 이메일에서 불필요한 공백을 제거하여 콘텐츠를 컴팩트하게 만듭니다.
여백 탐지
여백으로 인식할 최소 크기(px)입니다. 이 값보다 큰 빈 공간만 여백으로 탐지합니다. 값이 작으면 작은 여백도 탐지하고, 크면 큰 여백만 탐지합니다.
남길 여백
탐지된 상단/내부 여백에서 남길 비율(%)입니다. 30%이면 원래 여백의 30%만 유지합니다. 0%이면 여백을 모두 제거하고, 100%이면 원본 그대로 유지합니다.
페이지 구분선
페이지 사이에 연한 회색 구분선을 삽입합니다. 여백 줄이기가 켜져 있으면 사용할 수 없습니다.
